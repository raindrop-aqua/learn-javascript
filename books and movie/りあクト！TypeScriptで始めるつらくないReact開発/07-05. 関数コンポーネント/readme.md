

## アプリの環境設定と作成

```shell script
npx create-react-app jsx-function-component --typescript
```

アプリの環境設定と作成が終わったら、アプリケーションのフォルダに移動する

## Semantic UI React の設定
```shell script
yarn add semantic-ui-react semantic-ui-css
```

## アプリ開始

```shell script
yarn start
```
