## プロジェクト初期化

```shell script
npm init -y
```

## webpackのインストール

```shell script
# バージョンの確認
npm info webpack
# webpackのインストール
npm install --save-dev webpack@4.29.0

# バージョンの確認
npm info webpack-cli
# webpackのインストール
npm install --save-dev webpack-cli@3.2.1
```

## ライブサーバーのインストール

```shell script
## live server のインストール
npm install --save-dev live-server@1.2.1

## live server の実行
./node_modules/.bin/live-server

## live server の実行２
npx live-server
```
